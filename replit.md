# Tumbergia Azul - Digital Guide

## Overview
A static HTML website serving as a digital guest guide for "Tumbergia Azul" accommodation. The site provides visitors with information about the property, local attractions, restaurants, rules, WiFi access, and emergency contacts.

## Project Structure
- `index.html` - Main homepage with navigation grid
- `sobre.html` - About the property
- `wifi.html` - WiFi connection information
- `regras.html` - House rules
- `pontos-turisticos.html` - Local tourist attractions
- `restaurantes.html` - Restaurant recommendations
- `contato.html` - Contact information
- `emergencia.html` - Emergency contacts
- `dicas.html` - General tips
- `bem-vindos.html` - Welcome page
- `logo.png` - Property logo
- `server.py` - Python HTTP server for serving static files

## Technology Stack
- Pure HTML/CSS (no framework)
- Python 3.11 for serving static files
- Google Fonts (Inter, Playfair Display)

## Running the Project
The website is served via a Python HTTP server on port 5000:
```
python server.py
```

## Deployment
This is a static website that can be deployed as a static site.
